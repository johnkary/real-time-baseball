# Real-time Baseball

Follow a baseball game in real-time. Inspired by ESPN Gamecast.

## Technologies

* RethinkDB
* Node
* WebSockets

## Install

    $ git clone ...
    $ brew install rethinkdb
    $ brew install node
    $ npm install

## Setup

    $ rethinkdb
    $ node setup.js
    $ node server.js

## Run

<http://localhost:3000/>
